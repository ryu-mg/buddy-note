# 펫 MBTI 5문항 — v0 초안

Status: **DRAFT** — 사용자가 친구 강아지 3마리 관찰 후 실제 테스트 필요.
Target: 1-2분 온보딩. 각 문항 30초 이하.

## 설계 원칙

- 5문항 이상 금지 (이탈률 급등)
- 한 문항 = 한 성격 축 (겹침 금지)
- 선택지는 3-4개 (4개 기본, "기타" 없음)
- 문항은 "상황 vs 반응" 형태 — 추상 개념 금지 ("활발한가요?" X)
- 답을 LLM 프롬프트에 **자연어 형태 그대로** 주입 가능해야 함

## 성격 축 매핑 (5차원)

| # | 축 | 양극 | LLM 프롬프트 반영 |
|---|---|---|---|
| 1 | **에너지** | 활발 ↔ 차분 | 말투 속도, 느낌표 빈도 |
| 2 | **사회성** | 외향 ↔ 내향 | 낯선 사람/개 언급 톤 |
| 3 | **애착 스타일** | 껌딱지 ↔ 독립 | 가족 호칭, 거리감 |
| 4 | **자극 반응** | 대담 ↔ 겁많음 | 소리/낯선 물건 반응 묘사 |
| 5 | **루틴 선호** | 탐험 ↔ 반복 | 산책 루트 / 새 장소 태도 |

## 문항

### Q1. 에너지 — "산책 가자" 했을 때

```
A. 문 앞에 이미 도착해있다 (활발)
B. 꼬리만 흔들고 천천히 온다 (중간-활발)
C. 한 번 더 불러야 움직인다 (중간-차분)
D. 자고 있으면 산책보다 잠이 먼저다 (차분)
```

LLM 반영 예: A → "에너지 폭발, 문장에 느낌표 자주 / 말이 빠름"

---

### Q2. 사회성 — 공원에서 낯선 강아지가 다가올 때

```
A. 먼저 달려가서 인사한다 (외향)
B. 관심은 있지만 주인 뒤에서 살핀다 (중간)
C. 시선 돌리고 우리 갈 길 간다 (내향)
D. 짖거나 피한다 (경계형)
```

LLM 반영 예: D → "낯선 애들 얘기할 때 약간 긴장한 묘사, '나랑 거리 좀 둬' 느낌"

---

### Q3. 애착 스타일 — 가족이 화장실에 들어갈 때

```
A. 문 앞에서 운다/긁는다 (강한 껌딱지)
B. 문 앞에 앉아서 기다린다 (껌딱지)
C. 같은 방에서 잔다 (중간)
D. 소파에서 혼자 잘 논다 (독립)
```

LLM 반영 예: A → "가족 호칭 자주 쓰고 '엄마 어디 갔어' 같은 표현, 혼자 있던 시간 언급 많음"

---

### Q4. 자극 반응 — 청소기 (또는 드라이기) 소리가 나면

```
A. 일단 달려가서 짖는다 (대담-공격적)
B. 관찰한다, 천천히 다가간다 (탐구)
C. 멀찍이 자리 잡고 지켜본다 (신중)
D. 방에 숨는다 (겁많음)
```

LLM 반영 예: D → "예상 못한 소리/물건에 '깜짝 놀랐어' 톤, 작은 것도 크게 느낌"

---

### Q5. 루틴 선호 — 산책 루트를 바꾸면

```
A. 새 길에 더 신난다 (탐험)
B. 잠깐 헤매지만 금방 적응 (유연)
C. 원래 길 쪽으로 자꾸 끌어당긴다 (반복 선호)
D. 새 길은 거부하고 가만히 선다 (강한 반복 선호)
```

LLM 반영 예: C → "익숙한 장소/물건에 애착 강함, '맨날 가던 거기가 좋아' 톤"

## LLM 프롬프트 주입 포맷

5문항 답을 **1인칭 자기소개** 문장으로 변환해서 system prompt에 주입:

```
나는 {name}, {breed}야. 나를 한 줄로 말하면:
{Q1 결과 자연어} / {Q2 결과 자연어} / {Q3 결과 자연어} / {Q4 결과 자연어} / {Q5 결과 자연어}

예시 (마루, ENFP-like):
"나는 마루, 푸들이야. 에너지 폭발, 문 앞에 이미 도착해 있고 / 공원에서 먼저 달려가서 인사하고 / 엄마 화장실 앞에서 끙끙거리고 / 청소기는 일단 짖고 보고 / 새 길에 더 신나는 타입이야."
```

이걸 pet row 생성 시 `pets.persona_prompt_fragment` TEXT 컬럼에 저장 (캐싱). 매번 5개 답을 재조립하지 않음.

## Week 0 검증 계획

1. 친구 강아지 3마리(서로 다른 성격: 활발 외향 / 차분 내향 / 겁많은 중간) 주인에게 5문항 질문.
2. 응답 받아서 LLM 프롬프트 주입 → diary 1개씩 생성.
3. 주인에게 diary 보여주고 "우리 아이 같은가?" 1-5점 평가.
4. 평균 4.0+ = 합격. 그 이하면 문항 수정.
