# 펫 MBTI 4문항 — MVP v1

Status: **MVP 확정** — 강아지 전용. 사람 MBTI 코드(E/I, S/N, T/F, J/P)를 차용하되 질문은 강아지 행동 상황으로 묻는다.

## 설계 원칙

- 4문항 = 4축. 각 문항은 하나의 MBTI 글자를 결정한다.
- 선택지는 A/B 2개. 빠른 온보딩을 우선한다.
- 반려인 호칭은 별도 입력으로 받고, 일기에서 호칭으로 사용한다.
- 결과는 `{code} · {label}` 형태로 소개 확인 단계에 노출한다.

## 문항

### Q1. E/I — 처음 보는 친구가 다가올 때

```
A. 먼저 다가가서 냄새 맡고 인사한다 (E)
B. 반려인 곁에서 천천히 살핀다 (I)
```

### Q2. S/N — 새 장난감이나 새 장소를 만났을 때

```
A. 일단 궁금해서 탐험부터 한다 (N)
B. 익숙한 냄새와 물건을 먼저 찾는다 (S)
```

### Q3. T/F — 하고 싶은 것과 반려인 반응이 다를 때

```
A. 자기 페이스가 뚜렷하다 (T)
B. 반려인 표정과 목소리를 먼저 본다 (F)
```

### Q4. J/P — 하루 루틴이 바뀌었을 때

```
A. 정해진 시간과 순서를 좋아한다 (J)
B. 바뀐 흐름에도 금방 맞춘다 (P)
```

## LLM 프롬프트 주입 포맷

4문항 답을 `lib/pet-mbti.ts`에서 1인칭 자기소개 문장으로 변환해 `pets.persona_prompt_fragment`에 저장한다.

```
나는 {subject}{selectJosa(subject, "이야/야")}. {withJosa(companionRelationship, "은/는")} 내 반려인이야.
내 성격 유형은 {personalityCode} · {personalityLabel}.
나를 한 줄로 말하면:
{Q1 결과 자연어} / {Q2 결과 자연어} / {Q3 결과 자연어} / {Q4 결과 자연어}
```
