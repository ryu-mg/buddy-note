# buddy-note Copy Voice

## 원칙

- 버디노트의 일기는 강아지 1인칭 반말이다.
- 시스템 안내는 짧고 회복 액션을 같이 준다.
- 반려동물을 설명하지 말고, 버디가 직접 말하는 느낌을 우선한다.

## 기본 목소리

- `나`, `내가`, `내 하루`, `내 마음`을 쓴다.
- 보호자 대신 `반려인`을 쓴다.
- 앱 안 기록 명칭은 `버디노트`로 통일한다.
- 완료 순간은 `저장됐어`, `적어둘게`, `기억할게`처럼 버디의 행동으로 말한다.

## 에러와 fallback

- 사과보다 회복을 먼저 보여준다.
- 예: `내가 오늘 말이 잘 안 떠올랐어. 잠깐 임시로 적어둘게.`
- 예: `잠깐 길을 놓쳤어. 한 번만 더 불러줄래?`
- 버튼은 `다시 해볼래`, `조금 있다 다시 할래`처럼 낮은 압력으로 쓴다.

## 금지어

- `스마트`, `혁신`, `all-in-one`, `특별한 순간`, `완벽한 케어`
- 과한 감탄사, 이모지, 광고체 문장
- 사람 관찰 디테일을 특정하는 표현

## UI 카피 예시

- 홈 CTA: `사진 한 장 보여줄래?`
- 업로드 Step 1: `오늘 뭐 했어?`
- 업로드 Step 2: `사진 찍어둔 거 있어?`
- 업로드 Step 3: `{반려인 호칭}가 뭐 말해줄 거 있어?`
- 생성 중: `사진 들여다보는 중...`, `나 평소처럼 적어볼게`
